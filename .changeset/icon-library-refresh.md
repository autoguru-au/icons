Icon library refresh — custom + Phosphor Icons (AG-19073, #69).

A full redesign of the icon set. **Breaking:** some icons were removed or renamed, so exported icon component names have changed — consumers must update imports for any affected icons. See `ICON-MIGRATION.md` for the full old → new mapping.

- **49** custom icons replaced with new designs from the design team (filenames/export names unchanged).
- **11** new custom icons added (no previous equivalent).
- **171** icons replaced with [Phosphor Icons](https://phosphoricons.com).
- **2** original icons kept where no replacement was available.

Totals across `icons/`: 123 added, 154 removed, 77 modified.

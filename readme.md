# @autoguru/icons

### Install

```sh
yarn add @autoguru/icons \
    react
```

### Usage

```jsx
import { AccountBoxIcon } from '@autoguru/icons';
import { Icon } from '@autoguru/overdrive';

<Icon icon={AccountBoxIcon} size={16} />
```
